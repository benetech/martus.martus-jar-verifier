ซอฟต์แวร์ Martus(tm) รุ่น readme_verify_th.txt
------------------------------------------------------------

Benetech ปรับปรุงโปรแกรม Martus อย่างต่อเนื่อง
โปรดตรวจสอบรุ่นใหม่ล่าสุดได้ที่ http://www.martus.org และปฏิบัติตามคำแนะนำที่
ปรากฏบนเว็บไซต์ในการดาว์นโหลดซอฟต์แวร์มาร์ตัส โดยใช้โปรแกรม  SHA-1 sum 

ในอนาคต Benetech อาจเลือกที่จะแจกจ่ายรุ่นปรับปรุงของแฟ้ม martus.jar คุณไม่ควร
เชื่อใจแฟ้ม martus.jar ใด ๆ ที่ส่งไปให้คุณ จนกว่าคุณจะได้ตรวจสอบแล้วว่าเป็นของ
จริง อย่าได้ไว้ใจขั้นตอนการตรวจสอบอื่น ๆ ซึ่งจะทำให้คุณเข้าใจผิดได้

ใช้แต่โปรแกรมตรวจสอบที่พบบนซีดี Martus ที่ได้รับการยืนยันแล้วเท่านั้น
ดูส่วนที่มีหัวข้อว่า "Verifying a Martus jar file" บนหน้าดาวน์โหลด Martus สำหรับ
คำแนะนำในการตรวจสอบซอฟต์แวร์ Martus คำแนะนำดังกว่า สมมติว่าโปรแกรมตรวจสอบอยู่ใน
โฟลเดอร์ Verify ในซีดี Martus ถ้าคุณมีโปรแกรมตรวจสอบอยู่บนแผ่นดิสก์หรือบน
ฮาร์ดดิสก์ คุณต้องเปลี่ยนคำแนะนำให้สอดคล้องกับที่คุณมี


* ในวินโดวส์ (Windows):

เปิดหน้าต่างของ MS-DOS แล้วพิมพ์คำสั่ง 3 บรรทัดนี้ลงไป แทนที่ไดร์ฟซีดีรอมของคุณ
จาก "d" แล้วพาธของแฟ้ม .jar ที่คุณต้องการตรวจสอบ

d:
cd  \verify
ven  c:\path-to-file\martus.jar

หรือจาก Start menu เลือก Run แล้วพิมพ์:

d:\verify\ven  c:\path-to-file\martus.jar  

(เมื่อ d คือไดร์ฟซีดีรอมของคุณ และ path-to-file คือพาธของแฟ้ม .jar ที่คุณต้องการ
ตรวจสอบ)

ถ้าปรากฏ "Martus JAR verified." แสดงว่าแฟ้ม .jar นั้นถูกต้อง


* ในแมคโอเอส (Mac):

วางแฟ้ม .jar ใหม่ลงในโฟลเดอร์ Martus ที่อยู่แล้วบนฮาร์ดดิสก์ของคุณ แล้วรัน
โปรแกรมตัวสอบ:

cd  /Volumes
ls
cd  Martus (จากนั้นกดปุ่มแทบ เพื่อให้เติมชื่อเต็มโดยอัตโนมัติ)
cd  verify
java  -cp  .  JarVerifier  /Library/Java/Martus/martus.jar

ถ้าปรากฏ "Martus JAR verified." แสดงว่าแฟ้ม .jar นั้นถูกต้อง



* ในลินุกซ์ (Linux):

วางแฟ้ม .jar ใหม่ลงในโฟลเดอร์ ~/.Martus/ ที่มีอยู่แล้ว แล้วรันตัวตรวจสอบ
โค้ดต่อไปในสามารถวางใน bash หรือ sh เชลล์ได้ (แม้ว่าคุณต้องเป็น 'root' ในขณะที่
ต้องใช้คำสั่ง mount):

  mount=/mnt/cdrom
  [ -d $mount ] || mount=/cdrom		# for Debian and etc
  [ -d $mount/[vV]erify ] || mount -r $mount
  cd $mount/[vV]erify && java JarVerifier ~/.Martus/martus.jar

ถ้าปรากฏ "Martus JAR verified." แสดงว่าแฟ้ม .jar นั้นถูกต้อง

ถ้าคุณเห็น  bash: java: No such file or directory แสดงว่าคุณต้องเพิ่มโฟลเดอร์
bin ของ Java ลงใน PATH ของคุณ
ดูขั้นตอนที่ 1 ในขั้นตอนการติดตั้งบนลินุกซ์

ถ้าคุณเห็น Error loading class JarVerifier: Bad major version number แสดงว่า
คุณต้องติดตั้ง Java รุ่นใหม่กว่า และ/หรือ ใส่พาธของ Java รุ่นใหม่ที่สุดลงใน
PATH ของคุณ ดูขั้นตอนที่ 1 ในขั้นตอนการติดตั้งบนลินุกซ์
